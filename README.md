<p>A simple application for your outdoor running activities based on the MVVM architecture.
This project is developed with source code based on <a href="https://www.youtube.com/@PhilippLackner">Philipp Lackner</a> video course released in 2020.</p>

# PLANNING
<p>The Running App is one of the sports apps to help you control your outdoor runs. Everyone who wants to go in for sports is faced with a choice - what type of training to choose? The best option in terms of energy costs, efficiency and pleasure is running. And, of course, it is important to remember that in running, as in any other physical activity, you must adhere to regular classes. So the main role of this application is to provide the user with visual information about his runs, thereby motivating him to continue running.</p>
